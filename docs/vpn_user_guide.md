# Руководство пользователя VPN

## Как получить доступ
Обратиться к администратору и запросить VPN-доступ.

Администратор создаёт сертификат и конфигурацию.

---

## Установка клиента

### Windows
1. Скачать OpenVPN GUI
2. Установить
3. Поместить .ovpn файл в config
4. Подключиться

### Linux
```bash
sudo pacman -S openvpn
openvpn --config client.ovpn
