# Руководство пользователя VPN

Данное руководство описывает, как получить доступ к VPN и подключиться
на разных операционных системах.

## Получение доступа

Для подключения пользователь получает файл конфигурации `.ovpn`
от администратора системы.

Пользователь не создаёт сертификаты самостоятельно.

## Windows

1. Установить OpenVPN Client  
   https://openvpn.net/community-downloads/

2. Скопировать файл `user.ovpn` в папку:
   C:\Users\<username>\OpenVPN\config

3. Запустить OpenVPN GUI от имени администратора
4. Нажать Connect

## Linux

Установить OpenVPN:

sudo apt install openvpn

Подключение:

sudo openvpn --config user.ovpn

## macOS

Установить Tunnelblick:
https://tunnelblick.net/

Импортировать файл `.ovpn` и подключиться через интерфейс.

## Проверка подключения

После подключения меняется IP-адрес и появляется доступ
к внутренним ресурсам.
