# Описание инфраструктуры

Инфраструктура состоит из нескольких виртуальных машин,
каждая из которых выполняет свою роль.

## Компоненты

- CA — выпуск и подпись сертификатов
- VPN-сервер — приём подключений OpenVPN
- Monitoring — Prometheus и Grafana
- Logging — Loki
- Backup-host — хранение резервных копий

Все сервисы изолированы и взаимодействуют по сети.
